Over the last 3 years, Big Tech and high‑scale product companies have shifted mid‑to‑senior frontend machine coding rounds toward **architecturally rich, system‑aware implementations** rather than toy tasks. At the senior level, the interviewer cares less about “Does it render?” and more about:

- How state and data flow are structured (global vs. local, server vs. client).
- How rendering and interaction patterns scale (virtualization, async coordination, debouncing/throttling).
- How accessibility, performance (Core Web Vitals), error handling, and offline behavior are baked into the design.
